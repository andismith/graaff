    Creating a Blog

    Initial Setup

    1. Install Graaff using the instructions
    2. We're going to build a blog from scratch, so delete the contents of src/content folder
    3. Within the src/content folder create a folder called _data
    4. Copy the file logic/_data/default.json in to your content/_data folder and rename it site.json
    5. Edit the file and change the title of your site to the name of your blog.

    Creating the Index Page

    1. Create a file called index.hbs within src/content
    2. Give it the following parameters:
    ---
title: Home
section: Blog
layout: list.hbs
page: true

list:
  section: Blog
  start: 0
  length: 10
  sort: posted
  sortOrder: desc
---
